Custom Pipes Examples:

1. Simplest Pipe Example without paramter:

Square of {{valueForSquare}} = {{ valueForSquare | square }}

2. Pipe Example with one paramter:

{{valueForPower}} Power of {{valueForSquare}} Like {{valueForSquare}}{{valueForPower}} = {{ valueForSquare | power : valueForPower }}

3. Pipe Example with optional paramter:

{{valueForSquare}} DEFAULT PARAM i.e.. 3 = {{ valueForSquare | power }}

4. Pipe Example with multiple paramter:

{{valueForPower}} Power of {{valueForSquare}} + 20 Like ({{valueForSquare}}{{valueForPower}} + 20) = {{ valueForSquare | powerWithMultiParams : valueForPower : 20}}

5. Pipe Example with variable number of arguments:

//No parameter

{{ "Arun" | userFullName }}

// One parameter

{{ "Arun" | userFullName :"kumar"}}

// two parameter

{{ "Arun" | userFullName : "kumar" : "gudelli" }}
